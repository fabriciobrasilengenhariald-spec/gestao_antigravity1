# Gestão de Aluguel de Materiais

Sistema para gerenciamento de estoque e aluguel de materiais de engenharia.

## 🚀 Como Rodar em Outro Computador

Para continuar o desenvolvimento em uma nova máquina, siga este passo a passo:

1. **Pré-requisitos:**
   - Instale o [Node.js](https://nodejs.org/) (versão LTS recomendada).
   - Instale o [Git](https://git-scm.com/).

2. **Clone o Repositório:**
   ```bash
   git clone https://github.com/fabriciobrasilengenhariald-spec/gestao_antigravity1.git
   cd gestao_antigravity1
   ```

3. **Instale as Dependências:**
   ```bash
   npm install
   ```

4. **Configure as Variáveis de Ambiente:**
   - Crie um arquivo `.env.local` na raiz do projeto.
   - Copie o conteúdo de `.env.example` para `.env.local`.
   - Preencha as chaves (Supabase, Gemini, Telegram) com os valores corretos.

5. **Inicie o Projeto:**
   ```bash
   npm run dev
   ```

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
